# Release Checklist — OctoAcme

Use this checklist before any production release.

- [ ] Acceptance criteria for all stories in the release are met.
- [ ] All PRs in the release have passed CI, linting, and security scans.
- [ ] Release Engineer has validated the deployment pipeline and rollback plan.
- [ ] Observability/Oncall Lead has confirmed monitoring dashboards and alerts.
- [ ] Staging smoke tests completed and passed (list of smoke tests attached).
- [ ] Backup/snapshot completed if applicable.
- [ ] Communication scheduled and stakeholders notified (PM/Support/Product Lead).
- [ ] Post-deploy verification and monitoring window defined and owner assigned.
